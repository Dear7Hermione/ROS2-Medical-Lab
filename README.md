# ROS2 Medical Lab

A ROS2-based medical device simulation project designed for learning robotics software engineering and medical robotics systems.

## Goals

- Learn ROS2 Topics
- Learn ROS2 Services
- Learn ROS2 Actions
- Learn ROS2 Parameters
- Learn ROS2 Launch System

## Simulated Devices

- Patient Monitor
- Infusion Pump
- Alarm System
- Central Controller

## Technology Stack

- ROS2 Jazzy
- Python
- Ubuntu 24.04
